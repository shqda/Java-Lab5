# Задание. Интерфейсы матриц и графов

## Часть 1. Матрицы

Напишите интерфейс `Matrix` с несколькими реализациями:

- `UsualMatrix`
- `SquareMatrix` (расширяет `UsualMatrix`, из предыдущих заданий)
- `SparseMatrix` — для разреженных матриц

### Требования

- Все общие методы должны быть объявлены в интерфейсе `Matrix`
- `SparseMatrix` должен быть реализован с использованием `LinkedList`
  - при необходимости можно создать дополнительные вложенные (inner) классы

---

## Тестирование

Напишите программу, которая:

- Создаёт две случайные матрицы размером `1000 x 1000`
- В каждой матрице — `1000` ненулевых элементов
- Реализует их двумя способами:
  - через обычные матрицы (`UsualMatrix`)
  - через разреженные матрицы (`SparseMatrix`)

### Проверка

- Убедиться, что:
  - сложение матриц даёт одинаковый результат
  - умножение матриц даёт одинаковый результат

---

## Дополнительное задание

Создать интерфейс `Graph` с двумя реализациями:

- на основе **матрицы смежности**
- на основе **списка смежности**

---

### Методы интерфейса

- `connect(int vertex1, int vertex2)` — соединение вершин
- `disconnect(int vertex1, int vertex2)` — удаление ребра
- `equals()` — сравнение графов
- `toString()` — строковое представление

---

### DFS (поиск в глубину)

Реализовать в классе `Main` функцию:

- `dfs(Graph graph)`

- Функция должна:
  - принимать `Graph` в качестве параметра
  - выполнять обход в глубину (DFS)
  - выводить текущую вершину на каждом шаге

---

## Проверка

- Протестировать `dfs` на:
  - графе на основе матрицы смежности
  - графе на основе списка смежности
